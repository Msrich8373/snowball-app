# Snowball Income — веб-приложение

Трекер инвестиций и дивидендов. Точная копия Snowball Income для браузера.

## Функции
- Главная страница: стоимость портфеля, прибыль, IRR, пассивный доход
- Портфель: список активов с сортировкой и поиском
- Календарь дивидендов с выплатами по месяцам
- Аналитика: структура портфеля, рост дивидендов, диверсификация
- Несколько портфелей
- Добавление активов и сделок
- Тёмная тема (автоматически)

## Деплой на Vercel (бесплатно, 5 минут)

### Шаг 1 — GitHub
1. Зайди на [github.com](https://github.com) и зарегистрируйся (если нет аккаунта)
2. Нажми **New repository** → назови `snowball-app` → Create
3. Загрузи файлы `index.html` и `vercel.json` в репозиторий

### Шаг 2 — Vercel
1. Зайди на [vercel.com](https://vercel.com) → Sign up with GitHub
2. Нажми **Add New Project** → выбери репозиторий `snowball-app`
3. Нажми **Deploy**

Через 30 секунд получишь ссылку вида `https://snowball-app-xxx.vercel.app` — это твоё приложение!

## Локальный запуск
Просто открой `index.html` в браузере — всё работает без сервера.

## Добавление реальных данных
В файле `index.html` найди переменную `ASSETS` и замени демо-данные на свои активы.
